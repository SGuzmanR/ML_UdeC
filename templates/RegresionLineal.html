Modelo Regresion Lineal con Dataset Asignado

Utilizando el Dataset asignado (Ventas - Presupuesto en public), tiene dos columnas (X: Variable Independiente, Y: Variable Dependiente), el propósito es predecir Y a partir de X utilizando la Regresión Lineal.

El uso de la Regresión Lineal es ideal en este contexto, ya que hay una relación lineal entre ambas variables, este es útil cuando se quiere modelar la relación entre una variable dependiente y otra independiente.
