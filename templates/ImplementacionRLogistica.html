Implementación Regresion Logistica

Tomando el Dataset de Finanzas con las variables numéricas (X, Y). Para poder implementar la Regresión Logística, necesitamos una variable categórica como objetivo y variables numéricas, teniendo esto en cuenta, tenemos que convertir Y en una variable binaria compatible con el modelo de clasificación.

La Regresión Logística se basa en la probabilidad de que un evento ocurra dado un conjunto de variables independientes. Para la división del dataset, se utiliza train_test_split de Scikit-learn, que divide los datos en dos subconjuntos, uno para entrenar el modelo y otro para evaluar su desempeño.
