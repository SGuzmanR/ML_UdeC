Regresion Logística

Es un modelo estadístico utilizado para predecir el resultado de una variable dependientes binaria o categórica basada en una o más variables independientes. Predice probabilidades y usa la función logística (Sigmoide) para comprimir estas probabilidades entre 0 y 1.

P(y = 1 | X) = \(\frac{1}{1 + e^{-(\beta_0 + \beta_1 X_1 + \beta_2 X_2 + \dots + \beta_n X_n)}}\)
